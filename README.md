# Splash Screen com Animação em Flutter

Este projeto é um Splash Screen com animação de entrada e saída em Flutter, incluindo passagem de dados entre telas e uso de persistência simples.

---

## Tecnologias utilizadas

- Flutter
- VS Code
- Android Studio

---

### Efeitos e animações
- Opacidade / Transparência → `Opacity`
- Transformações → `Transform`
- Controle de animação → `AnimationController`

---

### Navegação
- Splash → Home
